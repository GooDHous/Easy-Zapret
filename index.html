Быстрая установка Zapret

Данное руководство поможет вам установить Zapret с помощью автоматического скрипта от YWTeam. Zapret - это инструмент для обхода блокировок интернета, основанный на оригинальных бинарниках Zapret с GitHub bol-van. Данным скриптом используется Zapret-discord-youtube от Flowseal

irm get.ywteam.ru | iex
Важная информация

Для успешной установки Zapret необходимо запустить PowerShell с правами администратора. Без этих прав скрипт не сможет выполнить все необходимые действия.

Пошаговая инструкция

1

Откройте PowerShell от имени администратора

Нажмите Win + X и выберите "Windows PowerShell (администратор)" Для Windows 11 "Терминал (администратор)" или найдите PowerShell в меню "Пуск", щелкните правой кнопкой мыши и выберите "Запуск от имени администратора".

2

Выполните команду установки

В открывшемся окне PowerShell введите команду:

irm get.ywteam.ru | iex

Нажмите Enter для выполнения команды.

3

Следуйте инструкциям установщика

Скрипт автоматически выполнит все необходимые действия:

  • Скачает последнюю версию Zapret от Flowseal
  • Установит программу в систему
  • Обновит список хостов для разблокировки
  • Откроет настройки автоматического запуска при старте системы
4

Проверьте работу Zapret

После завершения установки перезагрузите компьютер и проверьте доступ к ранее заблокированным сайтам, таким как YouTube, Discord и другим.

Часто задаваемые вопросы

Что делает Zapret?

Zapret обходит блокировки интернета, локально изменяя пакеты отправляемые вашим устройством. Это позволяет получить доступ к заблокированным сайтам и сервисам.

Безопасен ли этот скрипт?

Да, скрипт использует только оригинальный билд Zapret от Flowseal и не содержит вредоносного кода. Вы можете проверить код на GitHub создателя.

Как обновить Zapret?

Для обновления достаточно повторно выполнить команду установки. Скрипт автоматически скачает новую версию Zapret и обновит хосты.

Как удалить Zapret?

Для удаления Zapret выполните команду:

irm get.ywteam.ru/uninstall | iex
